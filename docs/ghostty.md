# ghostty 設定

kitty graphics protocol 対応のターミナルエミュレータ。nvim 内で画像/mermaid をプレビューするために使用。

## ファイル構成

```
ghostty/.config/ghostty/
└── config    # 設定ファイル (INI 風)
```

## 設定内容

- フォント: Cica Bold, 17pt
- カラースキーム: Oceanic Next
- `macos-titlebar-style = hidden`: タイトルバーなし
- `macos-option-as-alt`: Option キーを Alt として使用
